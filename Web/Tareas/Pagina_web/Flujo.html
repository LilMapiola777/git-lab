Flujo de Trabajo de Git y Github

Aquí se hablará del flujo de trabajo y los comandos git que utilizamos!

  1. Clonar el repositorio usando git clone "URL"

  2. Ver el estado con git status

  3. Crear un archivo o hacer un cambio

  4. Agregar los cambios y confirmarlos con git add y git commit -m "mensaje"

  5. Descargar los cambios con git pull origin main

  6. Subir los cambios con git push origin main
